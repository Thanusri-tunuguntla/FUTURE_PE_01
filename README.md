# FUTURE_PE_01
# AI Website Copy Generator for Local Businesses

## Business Chosen
Glow Style Salon – Visakhapatnam

## Project Objective
This project demonstrates how prompt engineering can be used to generate high-quality website copy for local businesses.

The prompt framework is designed to generate:
- Homepage copy
- Service descriptions
- Persuasive call-to-action sections

## Prompt Strategy

The prompt was designed to include:

Business Information
- Business name
- Location
- Services
- Target audience

Tone Control
The tone was set to "friendly and modern" to match the salon industry and appeal to young professionals.

Conversion Elements
The prompt includes CTA generation to encourage:
- Booking appointments
- Visiting the salon
- Scheduling bridal consultations

## Tools Used
Gemini AI

## Repository Structure

prompts/ → contains the structured prompts  
outputs/ → contains generated website copy  
screenshots/ → proof of prompt execution

## Outcome

Using structured prompts, AI generated website-ready content that includes:
- A strong homepage headline
- Clear value proposition
- Engaging introduction
- Persuasive call-to-action lines

This system can be reused for other local businesses like cafes, clinics, and coaching institutes.
